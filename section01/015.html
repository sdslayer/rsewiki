RSE-015

The Stress Ball That Informs

DANGER LEVEL: 2
CLEARANCE LEVEL: 3

A photo taken of RSE-015.

DESCRIPTION

RSE-015 is a stress ball that provides information to the user upon squeezing. The information it provides depends on how many times it has been squeezed. The facts it provides begin as fun trivia, but if squeezed too many times, the information it provides is so dark and soul-crushing that test subjects have been observed breaking down and falling into a deep depression that resulted in years of therapy for some and suicide for others. It is unknown exactly what information it provides, as all test subjects who have fallen into this state appear physically unable to relay it to others, almost as if their brain prevents them from doing so.

CONTAINMENT PROCEDURES

RSE-015 is to be held inside of a small locker within Location 01, and is to be only accessed by any classes higher than Level 3.
